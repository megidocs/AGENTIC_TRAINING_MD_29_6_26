# מרחב למידה בעברית

ברוכים הבאים לאתר לימודי קטן, ברור וקל לתחזוקה. האתר מיועד למורות, מורים ותלמידים שרוצים לפרסם שיעורים, דוגמאות ותרגולים באמצעות קבצי Markdown פשוטים.

{: .note }
אפשר להוסיף עמוד חדש בלי לגעת בעיצוב האתר: יוצרים קובץ Markdown, מוסיפים front matter, וכותבים את התוכן.

## מה יש באתר?

| אזור | מה נמצא בו |
| --- | --- |
| [אודות]({{ site.baseurl }}/about/) | מטרת האתר ואופן השימוש בו |
| [שיעור ראשון]({{ site.baseurl }}/lessons/lesson-1/) | דוגמה לשיעור עם טבלה ותיבת הדגשה |
| [שיעור שני]({{ site.baseurl }}/lessons/lesson-2/) | דוגמה לשיעור עם קטע קוד |

## דוגמת קוד

```text
lesson-title: "מבוא לכתיבה דיגיטלית"
file-name: lessons/new-lesson.md
```

האתר נבנה ומתפרסם אוטומטית באמצעות GitHub Actions ו-GitHub Pages.
