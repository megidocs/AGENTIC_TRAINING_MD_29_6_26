# מרחב למידה בעברית של עמית פלטי

מאגר זה מכיל אתר Jekyll קטן בעברית שמתפרסם באמצעות GitHub Pages ו-GitHub Actions. התוכן נכתב בקבצי Markdown, והעיצוב מגיע מהתמה Just the Docs בגרסה נעולה.

## הוספת שיעור חדש

צרו קובץ חדש בתוך `lessons/`, למשל `lessons/lesson-3.md`, והתחילו אותו כך:

```yaml
---
layout: default
title: "שיעור 3: כותרת"
parent: "שיעורים"
nav_order: 3
permalink: /lessons/lesson-3/
---
```

לאחר מכן כתבו את תוכן השיעור ב-Markdown רגיל.

## ניווט באתר

הניווט נקבע בעזרת front matter. השדה `nav_order` קובע את סדר העמודים. השדה `parent: "שיעורים"` מציב עמוד תחת אזור השיעורים. השדה `permalink` מגדיר כתובת יציבה לעמוד.

## פרסום עם GitHub Actions

כל דחיפה לענף `main` מפעילה את `.github/workflows/deploy-pages.yml`. התהליך בונה את האתר ומפרסם אותו ל-GitHub Pages. אם פרסום נכשל, פתחו את לשונית Actions במאגר, בחרו את הריצה שנכשלה, וקראו את שלב הבנייה או הפרסום שסומן באדום.

## שינוי גרסת התמה

התמה מוגדרת ב-`_config.yml` כך:

```yaml
remote_theme: just-the-docs/just-the-docs@v0.12.0
```

שנו גרסה רק בכוונה, לאחר בדיקת הערות השחרור של Just the Docs והרצת פרסום מוצלח ב-GitHub Actions.

## רישוי ותוכן מקורי

המאגר ציבורי, אך עצם הפרסום הציבורי אינו קובע רישיון שימוש חוזר לתוכן המקורי של המורה. אם נדרש רישיון לתוכן, יש להוסיף קובץ רישיון מפורש.

## תצוגה מקומית

תצוגה מקומית עם Ruby, Bundler או Jekyll היא אפשרות מתקדמת בלבד למי שכבר יש לו סביבת עבודה תקינה. היא אינה נדרשת לסדנה זו ואינה חלק מתהליך העבודה המחייב.
