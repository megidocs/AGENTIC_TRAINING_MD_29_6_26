# שיעור 1: פתיחה

בשיעור זה התלמידים מתנסים בקריאת הוראות קצרות, זיהוי מטרת משימה וכתיבת תשובה ברורה.

{: .note }
מומלץ לפתוח בדיון קצר: מה הופך הוראה לימודית לקלה להבנה?

## מטרות

| מטרה | תוצר צפוי |
| --- | --- |
| לזהות מילים מרכזיות בהוראה | סימון פעלים כמו "הסבירו", "השוו", "כתבו" |
| לנסח תשובה קצרה | פסקה בת 3-4 משפטים |
| לבדוק בהירות | שיפור ניסוח לפי משוב עמיתים |

## פעילות

1. קראו את ההוראה.
2. סמנו את הפעולה הנדרשת.
3. כתבו תשובה קצרה.
4. החליפו תשובות ובקשו משוב.
